
# 5-Day Weather Forecast
A simple application to display 5-day weather forecast using the OpenWeatherMap API.

### Demo
[http://weather-react-app.s3-website.eu-west-2.amazonaws.com/]


## Run
```
git clone
cd weather-forecast
npm i
```


## Start the dev server
```

npm run start

```

## Build
```

npm run build

```

## Test
```

npm run test

```
